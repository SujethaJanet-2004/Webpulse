# Webpuls

**Webpuls** is an AI-powered website analyzer that helps users understand their competitors’ websites by analyzing SEO, performance, and content structure.  

## Folder Structure
- `frontend/` → React app (user interface)
- `backend/` → Flask API (data processing)
- `scrapper/` → Website scraping and screenshot logic
- `docs/` → Notes, architecture plans

This project is being built step-by-step to learn full-stack + AWS development.
